# エコーブレード (Echo Blade)

**時間の残響と共闘し、刻一刻と変化する戦場を90秒間で制圧する"時間反響"アクション**

## 🎮 ゲーム概要

エコーブレードは、プレイヤーの斬撃が2秒後に「残響」として再現される独特なアクションゲームです。残響は敵にダメージを与えますが、自分に当たると危険。時間と空間を戦略的に活用し、90秒間の制限時間内でハイスコアを目指します。

### 🌟 主な特徴

- **残響システム**: 斬撃が2秒後に同じ位置で再現される
- **戦略的戦闘**: 残響を活用した戦術的な敵撃破
- **時間制限**: 90秒間の緊張感あふれるアクション
- **コンボシステム**: 連続撃破でスコア倍率アップ

## 🎯 操作方法

### 基本操作
- **移動**: ← → キー (A/D キー)
- **ジャンプ**: ↑ キー (W キー / スペースキー)
- **斬撃**: Z キー (マウス左クリック)
- **ダッシュ回避**: X キー (マウス右クリック)
- **ポーズ**: P キー (Esc キー)

### アクション詳細

#### 斬撃
- 前方120°扇形の攻撃範囲
- クールダウン: 0.4秒
- 2秒後に「残響」として同じ位置に再発動

#### ダッシュ回避
- 短距離の高速移動
- 無敵時間: 0.25秒
- 残響もすり抜け可能

#### 残響システム
- 斬撃後2秒で発動
- 敵にダメージを与える
- プレイヤーに当たるとノックバック＋時間減少
- 最大同時数: 8個

## 🏆 ゲームルール

### 勝利条件
- 90秒間生存してハイスコア獲得
- 最終ボス「タイムイーター」の撃破

### スコアシステム
- **基本撃破**: 100点
- **コンボボーナス**: コンボ数 × 5点
- **残響撃破**: +50%ボーナス
- **同時撃破**: 2体+200、3体+500、4体以上+1000

### 時間システム
- 制限時間: 90秒
- 敵撃破でわずかな時間延長
- タイムオーブ取得で+3秒
- 被弾で時間大幅減少

## 👾 敵キャラクター

### グリント（雑魚敵）
- **体力**: 1
- **行動**: 等速でプレイヤーを追跡
- **出現数**: 6-12体/ウェーブ

### クラッシュシェード（盾持ち）
- **体力**: 3
- **特徴**: 前面ガード
- **弱点**: 背後からのみダメージ有効

### タイムイーター（ボス）
- **体力**: 100
- **フェーズ1**: 床攻撃 & 中距離弾
- **フェーズ2**: 残響吸収 & 大範囲波動
